fastq2tsv

Format

cg fastq2tsv ?infile? ?outfile?

Summary

Converts data in fastq format to tab-separated format (tsv).

Arguments

infile
file to be converted, if not given (or -), uses stdin. File may be compressed.
outfile
write results to outfile, if not given, uses stdout

Category

Format Conversion
